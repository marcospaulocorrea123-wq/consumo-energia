## 📌 Sobre o Projeto

Este projeto foi desenvolvido como parte de um programa de iniciação em tecnologia.  
O objetivo é calcular o consumo mensal de energia elétrica de um aparelho, ajudando o usuário a entender melhor seus gastos.

---

## 🧠 Como funciona?

O programa solicita:

- Nome do aparelho
- Potência (em Watts)
- Tempo de uso diário (em horas)

E calcula o consumo mensal utilizando a fórmula:


consumoMensal = (potencia × horasDia × 30) / 1000


Também calcula o custo estimado com base em uma tarifa média de energia.

---

## ▶️ Como executar

1. Instale o Python
2. Baixe o projeto ou clone o repositório:

git clone https://github.com/seu-usuario/consumo-energia.git


3. Acesse a pasta:

cd consumo-energia


4. Execute o programa:

python app.py


---

## 💡 Exemplo de uso


Digite o nome do aparelho: Geladeira
Digite a potência (em watts - W): 150
Digite o tempo de uso diário (em horas): 10

Resultado:
Aparelho: Geladeira
Consumo estimado: 45.00 kWh/mês
Custo estimado: R$ 33.75/mês
